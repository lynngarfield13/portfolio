Air Combat Game

Game Developed in Python using Pygame

Beginning Screen

The user starts the game by clicking the Play button. The player's plane is controlled by using the up, down, left, and right arrows on the keyboard. Bullets can be fired by pressing the space bar.

Enemy Planes

There are 2 types of enemy planes. The small planes are killed with one bullet, the larger planes require 2 bullets to kill.

Health Bar

A health bar is provided to track the health of the player. When the health bar is depleted, the game ends.                      

Multiple Levels

The game has multiple levels. As the level increases, the speed of the enemy planes increases.

Game Over

The game ends when the player's health is depleted.
